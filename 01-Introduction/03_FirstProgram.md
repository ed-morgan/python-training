# How to write your first Python program


## Exercise

* Create a new folder

* Open an editor and create a file named main.py
* Write a print statement (```print("Hello World")```)
* Run following command ```python main.py```

## Counting Example


```python
count = 1

while count <= 10
    print(count)
    count = count + 1


if count == 1:
    print("Finished Counting")

 ```
